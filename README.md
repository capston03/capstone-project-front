### 기술 스택

---

안드로이드 버전을 확인해 주세요. 

**SDK 2.16.2 부터 3.0.0** 까지 지원합니다. 해당 버전에 맞지 않는 OS를 사용하여 실행하는 경우 오류가 발생할 수 있습니다.

다음 코드를 flutter project 폴더에서 실행해 주세요.

```
flutter pub run flutter_launcher_icons:main
```
